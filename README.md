# ecommerce-automation-testing-selenium
Selenium WebDriver automation project for testing SauceDemo e-commerce application using Java and TestNG.
Project Description

This project demonstrates automated testing of an e-commerce web application using **Selenium WebDriver with **Java and **TestNG.
The automation suite tests key functionalities of the **SauceDemo website including login, product management, cart operations, sorting, checkout process, and logout.

The objective of this project is to demonstrate end-to-end automation testing of a web application and validate core user workflows using Selenium.

Features Tested
User Login with valid and invalid credentials
Product listing verification
Add product to cart
Cart validation and product price verification
Product sorting (Price and Name filters)
Checkout workflow automation
Order confirmation validation
Logout functionality
Technologies Used
Selenium WebDriver
Java
TestNG
ChromeDriver
Eclipse IDE
Test Scenarios Covered
Login functionality validation
Adding products to cart
Product sorting using dropdown filters
End-to-end checkout process
Logout and session validation
